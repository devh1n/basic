수업내용 요약
html이란? Hyper Text Markup Language 의 약자로 태그는 약 150개 정도 있지만 평균적으로 한 웹페이지에 사용되는 태그의 수는 약 26개이다. 모든 태그를 암기하는것 보다는 기초적인 태그를 완벽하게 이해하고 여러상황에서 유연하게 활용하는 것이 중요하다.
strong - 글자를 굴게 표시
u - underline의 u 밑줄을 표시
h1 - 제목을 표시, 크고 굵은 글자로 나타나며 아래에 빈줄이 생성된다. 1~6까지 가능하고 숫자가 커질수록 글자는 작아지는 반비례관계이다.
br - 줄을 바꿀 수 있다. 특징적으로 />을 사용하지 않는다.
p - paragraph의 p 단락
margin - css에서 코드 사이의 여백을 나타내줌
img - 이미지 표시
src - source의 src
width - 넓이
li - list의 li, 반드시 부모 태그를 가지고있음
ul - unordered list, 자식태그있음
ol - ordered list, 자식태그있음
title - 상단 탭에 나타나는 제목, 본문의 제목
meta - 저장방식
char - 문자
set - 규칙
body - 본문
head - 제목
html - head와 body를 묶는 최고위층 태그
!doctype html - 관용적으로 맨 위에 쓴다
a - anchor의 a, 닻, 하이퍼링크를 걸때 사용
Hyper Text - 링크
ref - reference의 ref, 참조
